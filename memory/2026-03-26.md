## 任务1：OpenClaw最新玩法、技巧、配置优化方案学习

### 1. 核心配置优化（2026.3版本新功能）
- **多账号飞书配置**：支持同一个飞书开放平台下多个应用在不同频道使用，通过 `channels.feishu.accounts` 配置多账号，指定defaultAccount
- **记忆插件体系**：支持 mem0、memory-core、memory-lancedb 多种向量记忆后端，可在 `plugins` 配置启用
  - memory-core：OpenClaw官方推荐，本地LanceDB存储，内置意图隔离防记忆污染
  - mem0：云端同步方案，适合多设备访问
  - memory-lancedb：社区维护版，更多自定义选项
- **自动会话重置**：2026.3版支持每日凌晨自动重置会话，保持上下文清爽，避免累积膨胀

### 2. 安全性优化
- `gateway.bind` 应设置为 "127.0.0.1"（绝对不要设置为0.0.0.0）
- 应修改默认端口（18789）为随机高端口以提高安全性
- API密钥应存储在 `models.providers` 中，绝对不要提交到git

### 3. 性能优化
- 使用 qmd 预处理插件可大幅加速长文本处理
- 简单任务路由到本地模型，重推理任务路由到云端
- 启用 heartbeat 功能进行每日检查和自动更新

### 4. 自动化技巧
- 使用 cron 定时任务进行定期研究/报告任务
- `feishu-evolver-wrapper` + cron watchdog（每10分钟执行ensure）保证进化进程稳定运行
- PowerShell换行技巧：使用 `;` 分隔多条命令，避免使用 `&&`（PowerShell语法不兼容）
