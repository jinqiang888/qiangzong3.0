## 三、今日凌晨学到的实用技巧

### 1. **文件名中文编码技巧（Windows PowerShell）**
- **问题**：中文文件名在PowerShell中直接引用经常路径找不到
- **解决方案**：用通配符匹配/管道传递，避免直接输入中文路径
  ```powershell
  # 不好用
  Get-Content 2026-03-17学习汇报.md

  # 更好用
  Get-ChildItem *.md | Sort-Object LastWriteTime -Descending | Select-Object -First 1 | ForEach-Object { Get-Content $_.FullName -Encoding UTF8 }
  ```
