## 记忆系统架构

### 第一层：会话记忆（临时）
- 当前对话 context window (200K tokens)
- 会话文件保留在 sessions 目录
- 不跨会话保留

### 第二层：文件记忆（永久）
- `MEMORY.md` — 长期精炼记忆，每次主会话自动加载
- `memory/YYYY-MM-DD.md` — 每日原始日志
- `~/.agents/memory/` — 扩展分类记忆（projects/people/knowledge 等）
- `~/.self-improving/` — 自我改进记忆（corrections/lessons）

### 第三层：语义搜索（增强检索）
- 基于 OpenAI text-embedding-3-small
- 命令：`openclaw memory index` / `openclaw memory search`
- `openclaw memory status --deep` 查看状态

### 防错乱机制
- 主会话才加载 MEMORY.md
- 群聊/子会话不读取个人记忆
- 重要信息立即写入文件，不依赖"记住"
- 定期维护：合并日志 → MEMORY.md，清理过期内容
