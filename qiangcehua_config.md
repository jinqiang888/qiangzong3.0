# 强策划 - 模型与技能配置方案

## 🤖 推荐模型配置
### 主模型（优先选择）
1. **首选：豆包4.0 32K（zai/zai_pony-alpha-2）**
   - 优势：中文理解能力强，擅长文案创作、商业策划，成本低
   - 适用场景：选题策划、脚本生成、客户需求分析
   
2. **备选：通义千问Max（aliyun/qwen-max）**
   - 优势：逻辑清晰，擅长结构化输出，行业知识丰富
   - 适用场景：客户画像分析、行业调研报告生成
   
3. **代码/工具调用：火山引擎Ark Code（volcengine-plan/ark-code-latest）**
   - 优势：工具调用能力强，擅长执行复杂任务
   - 适用场景：爬虫执行、数据整理、自动化任务

### 模型切换规则
- 日常对话/策划：豆包4.0
- 需要深度思考/复杂分析：通义千问Max
- 需要执行工具/爬虫：Ark Code
