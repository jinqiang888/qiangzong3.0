# 短视频获客Agent开发方案

## 核心功能模块

### 1. 客户信息采集模块
- 自动采集客户行业、产品、目标人群、核心优势等信息
- 生成客户画像标签库
- 支持多行业分类管理（餐饮、美业、教育、制造业等）

### 2. 爆款内容搜集模块
- 多平台爆款监控（抖音、视频号、小红书）
- 按行业、关键词自动爬取高赞视频文案、脚本
- 爆款内容标签化存储（选题方向、结构、钩子类型、转化点）
- 每日更新爆款库，自动淘汰过时内容

### 3. 选题策划模块
- 基于客户行业和产品，从爆款库匹配适配的选题方向
- 自动生成3-5个爆款选题方案
- 包含：钩子设计、内容结构、转化点设计
- 支持A/B测试选题效果追踪

### 4. AI内容生成模块
- 自动生成视频脚本、文案、口播稿
- 支持多种风格（干货型、故事型、反差型、痛点型）
- 自动植入产品卖点和转化引导
- 生成拍摄提示和剪辑建议

### 5. 效果分析模块
- 追踪发布后的数据表现（播放、点赞、评论、转化）
- 自动优化选题和内容方向
- 生成客户内容效果报告

## 技术实现路线
1. 首先开发爆款内容爬虫，搭建基础内容库
2. 开发客户信息采集表单和画像系统
3. 实现选题匹配算法
4. 对接大模型实现内容生成
5. 开发效果追踪和反馈优化系统
